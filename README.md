> 一个移植 https://github.com/nexmoe/hexo-theme-nexmoe 的 typecho 主题

## 安装
按照常规 Typecho 主题安装即可

 1.解压后重命名主题文件夹（文件夹名称不能带横杠）
 2.移动到 /usr/themes/ 文件夹中
 3.到后台启用主题
